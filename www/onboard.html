Free tier — quick start

No account needed. Add to your MCP client config and start querying.

1
Add the server to your MCP client
Open your Claude Desktop / Cursor / Windsurf config and add one or more servers.
2
Paste the config snippet
Example for ARES (company registry) and DD (due diligence):
claude_desktop_config.json / mcp.json
{
  "mcpServers": {
    "ares": {
      "url": "https://ares.cz-agents.dev/mcp"
    },
    "dd": {
      "url": "https://dd.cz-agents.dev/mcp"
    },
    "sanctions": {
      "url": "https://sanctions.cz-agents.dev/mcp"
    }
  }
}
3
Restart your client and ask a question
Try: "What can you tell me about IČO 27074358?" — the AI will call the tools automatically.

Free tier is rate-limited at 30 requests/IP/day. For production use or higher quotas, see paid plans.

Paid tier & trial — setup

After subscribing or receiving your trial token, connect it to your MCP client.

1
Get your token
Paid: After checkout you'll receive your API token by email and in your billing dashboard.
Trial: Request a trial — token delivered by email within 1 business day.
2
Add the token to your MCP config
Replace czat_your_token_here with your actual token:
claude_desktop_config.json — API Compliance tier
{
  "mcpServers": {
    "ares": {
      "url": "https://ares.cz-agents.dev/mcp"
    },
    "dd": {
      "url": "https://dd.cz-agents.dev/mcp",
      "headers": { "Authorization": "Bearer czat_your_token_here" }
    },
    "sanctions": {
      "url": "https://sanctions.cz-agents.dev/mcp",
      "headers": { "Authorization": "Bearer czat_your_token_here" }
    },
    "ddplus": {
      "url": "https://ddplus.cz-agents.dev/mcp",
      "headers": { "Authorization": "Bearer czat_your_token_here" }
    }
  }
}
3
Test your connection
Restart your client and ask: "Run a full due diligence report on IČO 27074358."
You should see nominee check, risk timeline, address crowding, and linked entities.
4
Manage your subscription
View usage, rotate tokens, update billing in your billing dashboard.
