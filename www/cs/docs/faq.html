FAQ

Jaký je rozdíl mezi MCP a REST?
MCP je nástrojově orientované a funguje dobře uvnitř Claude Desktop, Cursor a agent runtimeů. REST je URL-orientované a hodí se pro skripty, batch kontroly a interní systémy.
Potřebuji token pro základní použití?
Ne. Free hostovaný REST i MCP přístup funguje bez tokenu, s limitem per IP. Placené tokeny odemykají vyšší měsíční kvóty a plné DD funkce.
Co obsahuje DD report?
DD report kombinuje firemní fakta, sankční kontrolu, insolvenční signály, statutární kontext, rizikové skóre a red flags. Free DD je základní; placený DD zahrnuje plnou hloubku reportu.
Jak aktuální jsou data?
ARES je cachován 1 hodinu na straně serveru. ISIR je kontrolován live. Sankční data jsou aktualizována týdně.
Mohu to použít pro KYC/AML?
Ano, jako screeningová a důkazní vrstva. Za interní politiku, právní přezkum a finální KYC/AML rozhodnutí odpovídáte vy.
Co je to statutární řetězec / detekce UBO?
Jde o heuristické procházení statutárních orgánů a propojených entit. Pomáhá odhalit vztahy, ale nenahrazuje výpis z oficiálního registru skutečných majitelů.
Co se stane, když překročím kvótu?
API vrátí 429. Odpovědi s rate limitem obsahují příslušné hlavičky a mohou obsahovat Retry-After.
Kde získám token / trial?
Použijte formulář nebo tlačítka na stránce ceník. Tokeny jsou doručeny po onboardingu nebo po checkout.
Mohu si to self-hostovat?
Ano. Open-source MCP servery lze self-hostovat pod licencí projektu. Hostované plány přidávají spravovanou kvótu, tokeny a placené DD funkce.
