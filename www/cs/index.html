Claude Desktop / Cursor

Přidej do claude_desktop_config.json:

{
  "mcpServers": {
    "ares": { "command": "npx", "args": ["-y", "@czagents/ares"] },
    "cnb":  { "command": "npx", "args": ["-y", "@czagents/cnb"] }
  }
}

Remote HTTP (bez npm)

{ "mcpServers": {
    "ares": { "url": "https://ares.cz-agents.dev/mcp" },
    "cnb":  { "url": "https://cnb.cz-agents.dev/mcp" }
}}

Příklad

User: Jaká je DIČ a sídlo firmy Seznam.cz? Přepočti 100 EUR na CZK.

Agent → lookup_by_ico(ico="26168685")
Agent → convert(amount=100, from="EUR", to="CZK")

Seznam.cz, a.s., DIČ CZ26168685, Radlická 3294/10, Praha 5 Smíchov.
100 EUR = 2436.00 CZK (kurz ČNB 2026-04-15).
