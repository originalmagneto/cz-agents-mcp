REST API

Stejná data jako MCP, ale plain GET requesty. Free endpointy nepotřebují auth — stačí curl. Placené volání přidávají Authorization: Bearer czat_.... → Kompletní REST API reference

# vyhledání firmy (zdarma)
curl https://ares.cz-agents.dev/v1/companies/27074358

# sankční kontrola (zdarma)
curl "https://sanctions.cz-agents.dev/v1/sanctions/check?ico=27074358"

# full DD report (placené)
curl -H "Authorization: Bearer czat_..." https://dd.cz-agents.dev/v1/dd/27074358

Response envelope: {"data": {...}, "meta": {"fetched_at": "...", "source": "ares"}} — viz všechny endpointy.

Datové principy

cz-agents používá pouze státem garantované veřejné registry: ARES (MF), ČNB, ISIR (MSp), EU FSF, OFAC SDN. Nekupujeme od paywally profesních komor (Centrální evidence exekucí Exekutorské komory) ani od komerčních přeprodejců (Cribis, Bisnode). Veřejná data mají zůstat zdarma v místě použití — tvoje předplatné platí údržbu, ne přístup k datům, která stát už publikuje.

Ceník

ARES a ČNB jsou zdarma, bez omezení pro všechny. Sankce a DD mají free tier s denním limitem; placené tiery limit ruší a přidávají funkce (alerty, batch, statutární řetězec). Zrušíš kdykoli.

Příklad

Smyšlená data — IČO i údaje níže jsou fiktivní, slouží jen k ukázce tvaru odpovědi.

User: Udělej due-diligence report na IČO 12345678.

Agent → get_dd_report(ico="12345678", depth="full")
        ↳ ARES + EU/OFAC sankce + screen statutárů

Acme Imports s.r.o. — Pražská 100, Praha 1, vznik 2018.
Plátce DPH (CZ12345678). Statutář: Jan Novák (od 2023-04-12).
Sankce: BEZ MATCHE (0/24 856 záznamů).
Osobní insolvence Jana Nováka: bez záznamu.
Bydliště na úřadu: bez záznamu (běžná ulice).
Předchozí firmy v konkurzu (dle příjmení): bez záznamu.
Risk score: 12/100 (low) — flag: chybí transparentní účet DPH.
